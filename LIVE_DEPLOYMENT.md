## ğŸ› Troubleshooting

### Data Not Persisting
- **Issue**: Activities disappear after refresh
- **Solution**: Check if cookies/storage is enabled in browser settings
- **Solution**: Ensure IndexedDB is not blocked

### Charts Not Displaying
- **Issue**: Progress charts don't show
- **Solution**: Check browser console for errors (F12)
- **Solution**: Verify Chart.js CDN is accessible

### Slow Loading
- **Issue**: Site loads slowly
- **Solution**: Check internet connection
- **Solution**: Clear browser cache
- **Solution**: Netlify CDN should make it fast globally

### Export Not Working
- **Issue**: Export button doesn't download file
- **Solution**: Check browser download settings
- **Solution**: Ensure pop-ups are not blocked
